这是一个Vue的入门教程

这个 Demo 主要讲述 Vue 的计算属性

计算属性表现上和 Vue 实例的属性、方法一样，也是通过表达式输出结果，虽然 Vue 的指令和属性表达式支持各种计算，但是为了 html 部分看起来不那么复杂，这里推荐大家对于计算复杂的属性，应当使用计算属性

同时计算属性有一个特别重要的特性，那就是结果缓存。假设有一个计算需要消耗2s（夸张了），如果我们用表达式的话，那么每次读取这个值，都要消耗2s，但是用计算属性后就不一样了，第一次仍然消耗2s，但第二次读取时，就会直接从缓存中读取了

{{time()}}

{{time()}}

通过计算属性，time 的值被缓存了，第二次读取时，直接读取了上一次的值，所以两个值永远一致
